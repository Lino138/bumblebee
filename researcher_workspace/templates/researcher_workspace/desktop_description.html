Researcher Desktop

A Personal Computer in the cloud that you can use as an extra compute resource for your research. You may find this useful to free up your laptop/workstation or to have a computer that you can leave running for long periods of uninterrupted processing. The Researcher Desktop runs on the Melbourne Research Cloud and uses the same University of Melbourne Windows or Linux (Ubuntu) desktop environment and applications that you are probably already using.

Specifications:

  • Cloud Virtual machine - 4 Cores, 16GB RAM, 200 GB harddisk (temporarily expandable to 16 cores, 64GB RAM for 7 days at a time)
  • OneDrive and UoM network share access
  • Linux (Ubuntu) or Windows 10
  • Remote desktop access
  • RStudio, Jupyter And Matlab preinstalled, as well as having Software Centre to install other applications

You may use this for the duration of your research.
