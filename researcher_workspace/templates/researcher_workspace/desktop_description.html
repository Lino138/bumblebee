ARDC Nectar Research Cloud Virtual Desktop Service

The content on this page requires updating.

This service allows you to easily and quickly create a Virtual Desktop - a personal computer in the cloud that you can use as an extra compute resource for your research. You may find this useful to free up your laptop/workstation or to have a computer that you can leave running for long periods of uninterrupted processing. The Virtual Desktop Service runs on the ARDC Nectar Research Cloud.

Specifications:

  • Cloud Virtual machine - 4 Cores, 16GB RAM, 200 GB harddisk (temporarily expandable to 16 cores, 64GB RAM for 7 days at a time)
  • OneDrive and UoM network share access
  • Linux (Ubuntu) or Windows 10
  • Remote desktop access
  • RStudio, Jupyter And Matlab preinstalled, as well as having Software Centre to install other applications
